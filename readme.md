# Proyecto de aprendizaje

El propósito de este proyecto es:
- Aprender a usar el preprocesador SASS
- Practicar el uso de características nuevas de CSS
- Practicar el uso de diferentes técnicas para crear elementos gráficos
- Practicar la creación de animaciones simples con transition y complejas con Keyframes
- Practicar el desarrollo responsivo Desktop-first
